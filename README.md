## 按语   
围绕AI相关的问题，是目前发展相当重要的问题。  
很多问题，人类社会已经探讨、发展了上千年，积累了相当的经验和方法。  
但AI似乎不同：
- 它太新，新到几十年的AI发展与千万年的人类文明积累不对等。
- 它太快，快到几年间其发展就已不可同日而语。
- 它太有影响，极具潜力地影响到可见与不可见的未来，且无法预见任何休止之处。
- 它不可回撤，它具备影响现实世界的巨大潜力，这种发展动能固然可以调整，但AI这个议题一旦登录历史舞台，似乎就已不可撤销。

## 议题  
在这个项目中，核心议题只有一个：如何用历史的眼光，在绚烂的人类文明的长流中，找到属于AI的位置？  
诚然核心议题如此，但这不避免的会挑战到过去的许多未曾言明的问题，如今也许已不可回避。
